3. controle de pedidos em uma lanchonete
tabelas:
produto: nome, descricao, preco.
pedido: lista de produtos, valor total, data, status (aberto/finalizado).

Regras de negocio:
Proibido cadastrar produtos com preco negativo ou nome vazio.
Pedido so pode ser finalizado se tiver pelo menos um produto adicionado.
calculo automatico do valor total do pedido com base nos produtos.
consulta dos pedidos realizados em determinado dia.
saber quanto foi faturado naquele periodo.
